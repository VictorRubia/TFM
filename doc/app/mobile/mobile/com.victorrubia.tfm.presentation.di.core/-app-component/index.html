AppComponent

@Singleton()
@Component(modules = {AppModule.class, NetModule.class, DataBaseModule.class, UseCaseModule.class, RepositoryModule.class, RemoteDataModule.class, LocalDataModule.class, CacheDataModule.class})
public interface AppComponent

AppComponent interface is the core component of the application and all subcomponents will be scoped to it. Since all subcomponents depend on this component, it will be the entry point for all. Singleton scope is used to ensure that all subcomponents created by this component are unique. This component will be used by Dagger to create all subcomponents.
